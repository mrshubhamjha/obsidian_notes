**1) single quote and double quote both allowed**

```
console.log('hello world')

console.log("hello world")
```
  

2) **backticks (``). also allowed**
  

```
console.log(`hello world`)
```

3) **semicolon is good practice and should be implemented**

  
```
console.log("hello world");
```
